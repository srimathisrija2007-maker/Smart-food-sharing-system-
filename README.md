# 🍱 Smart Leftover Food Sharing System ♻️

## 📌 Problem Statement
Food waste is a major issue in colleges, hostels, and events. Large amounts of edible food are wasted daily, while many people struggle to access basic meals. There is no simple system to connect food donors with people in need.

---

## 💡 Solution
The Smart Leftover Food Sharing System is a simple Python-based application that helps reduce food waste by connecting people who have extra food with those who need it.

---

## ⚙️ Features
- Add leftover food details (name, quantity, location)
- View available food items
- Collect food easily
- Simple and user-friendly system

---

## 🛠️ Tech Stack
- Python
- Lists
- Functions

---

## 🧠 How It Works
1. User adds food details
2. Data is stored in a list
3. Users can view available food
4. Food can be collected and removed from the list
